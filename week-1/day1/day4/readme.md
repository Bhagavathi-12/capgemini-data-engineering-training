# 📊 SQL & PySpark Practice Repository

## 📌 Overview
This project contains practice notebooks focused on SQL concepts and PySpark transformations.  
It helps in building strong fundamentals required for data engineering and analytics roles.

---

## 📁 Project Structure

.
├── Regex queries.ipynb
├── StringFunctions.ipynb
├── Datefunctions.ipynb
├── NULLIF.ipynb
└── README.md

---

## 📚 Files Description

### 🔹 Regex queries.ipynb
- Covers Regular Expressions (REGEX)
- Extract numbers, alphabets, and patterns
- Email and phone parsing
- Useful for data cleaning

### 🔹 StringFunctions.ipynb
- String operations:
  - SUBSTRING
  - LENGTH
  - UPPER / LOWER
  - TRIM
- Helps in formatting and transforming data

### 🔹 Datefunctions.ipynb
- Date-related functions:
  - Current date/time
  - Formatting dates
  - Date arithmetic
- Useful for time-based analysis

### 🔹 NULLIF.ipynb
- Handling NULL values
- Using NULLIF for conditional logic
- Preventing errors like division by zero
